# Borradores de Hyprland

Esta carpeta contiene configuraciones escritas desde el Mac que todavía no han sido probadas en Arch Linux.

Los archivos aquí no se enlazan ni se copian automáticamente a ~/.config/hypr.

Flujo:

1. Escribir o modificar un borrador.
2. Guardarlo mediante Git.
3. Probarlo posteriormente en Arch.
4. Moverlo a .config/hypr solo después de verificarlo.
