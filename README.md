# LinuxPC Dotfiles

Configuraciones reproducibles de mi entorno Arch Linux y Hyprland.

## Estructura

- `.config/hypr`: configuración de Hyprland.
- `.config/waybar`: barra de estado.
- `.config/kitty`: terminal Kitty.
- `.config/rofi`: lanzador de aplicaciones.
- `.local/bin`: scripts personales.
- `docs`: documentación del proyecto.

## Objetivos

- Mantener las configuraciones bajo control de versiones.
- Poder editar desde el Mac y sincronizar con Arch Linux.
- Documentar cada cambio importante.
- Facilitar la restauración y automatización del sistema.
